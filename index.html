API-REST

Este proyecto es una API de productos desarrollada en Node.js que se conecta a una base de datos MongoDB. Esta API proporciona endpoints para realizar operaciones CRUD (Crear, Leer, Actualizar, Borrar) en una colección de productos. Esto incluye la capacidad de crear nuevos productos, listar todos los productos, eliminar productos existentes, actualizar información de productos y realizar búsquedas avanzadas.

    ecommerce MVC

    Este proyecto es una tienda en línea que ofrece una experiencia fluida de compra. Los usuarios pueden iniciar sesión y agregar productos al carrito de compras sin necesidad de recargar la página, gracias a la implementación de la tecnología Fetch. La arquitectura del proyecto sigue el patrón Modelo-Vista-Controlador (MVC), lo que garantiza una organización eficiente del código y una separación clara del codigo entre los diferentes componentes.

    Technology Stack:

  • JS
  • PHP
